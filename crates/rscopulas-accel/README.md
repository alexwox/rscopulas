# rscopulas-accel

`rscopulas-accel` is the internal acceleration crate for the `rscopulas` workspace. It provides CPU-parallel helpers, runtime backend capability detection, and the narrow CUDA and Metal kernels currently used by `rscopulas-core`.

Most users should depend on `rscopulas-core` instead of using this crate directly.
